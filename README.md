# 🛒 Shopping Application (Python CLI Project)

## 📌 Description
This is a simple menu-driven shopping application built using Python.
It allows users to select products, enter quantity, and get final bill with discount.

## 🚀 Features
- Electronics, Clothing, Grocery categories
- Product selection with price
- Quantity input
- Discount system:
  - 20% if total > 30000
  - 10% if total > 15000
- Final bill calculation

## 🛠 Technologies Used
- Python
- Command Line Interface

## ▶️ How to Run
1. Install Python
2. Run:
   ```bash
   python shopping_application.py
